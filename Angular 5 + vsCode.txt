 1º Instalar node: https://nodejs.org/en/
 2º Instalar Angular Cli: npm -g @angular/cli (Rodar comando no cmd)
 3º Instalar Ide vsCode: https://code.visualstudio.com/
 4º Instalar plugin: vscode-icons
 5º habilitar plugin: CTRL + SHIFT + P procurar por Theme Icon
 6º Instalar plugin: auto-import
 7º Instalar plugin: Angular v5 Snippets(by john Papa)
 8º Instalar TypeScript: npm install -g typescript (Rodar comando no cmd)
 
 ############## COMANDOS ANGULAR #############
  - Criando novo projeto: ng new nome
  - Criando novo componente: ng g component nome
  - Start projeto: ng serve
  - Criando serviço: ng g service nome
